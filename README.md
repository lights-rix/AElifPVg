# 项目简介

## 核心功能
- **前后端代码及SQL文件**
- **万字文档**
- **参考导入调试视频**
- **讲解视频**
- **开发工具安装包**

## 适用场景
- 学生群体交流学习
- 教育机构内部资料分享
- 学习小组组建与活动
- 学习论坛互动

## 技术特点
- **架构**: B/S、MVC、Web框架
- **开发环境**: IDEA、JDK1.8、Mysql、Maven、Navcat
- **技术栈**: Java、SpringBoot、Vue
